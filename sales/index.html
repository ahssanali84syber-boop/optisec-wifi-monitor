FAQ

Common Questions

What hardware do I need? ▶
You need a Linux machine running BlackArch (or Arch-based distro), and a WiFi adapter that supports monitor mode — such as the Alfa AWUS036ACH. A secondary interface for internet connectivity is recommended.
Is a Groq API key included? ▶
No — Groq offers a generous free tier. Sign up at groq.com, generate a free API key, and paste it into the config wizard (python main.py config). The tool works fully offline without one.
Can I use this for penetration testing? ▶
Optisec is a passive defense tool — it monitors and alerts, it does not attack. It is suitable for authorized network security audits, home lab monitoring, and corporate WiFi defense. Always ensure you have authorization to monitor any network.
What is the license bound to? ▶
The license key is generated from your machine ID and your name on first run — it is stored in ~/.optisec/license.key. One license = one machine. Contact support to transfer to a new device.
Do I get future updates? ▶
Yes. One-time payment includes all future v1.x updates. Major version upgrades (v2.0+) may be offered at a discounted upgrade price for existing customers.
